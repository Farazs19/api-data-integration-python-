
# API Data Integration Project

## What This Project Does
This Python project automatically fetches live data 
from an external API, cleans it, and saves it to Excel.

## Tools Used
- Python
- Requests Library
- Pandas Library
- Excel (Output)

## How it Works
1. Python sends GET request to API
2. Receives JSON data
3. Cleans data using Pandas
4. Saves to Excel file

## Output
- api_data.xlsx - Contains fetched and cleaned data
